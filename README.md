# ⚽ 2022 World Cup Data Analysis: Uncovering Modern Football Trends

## 📖 프로젝트 개요 (Project Overview)
이 프로젝트는 **2022 카타르 월드컵**의 실제 경기 데이터를 분석하여 현대 축구의 트렌드를 파악하고, 국가별 퍼포먼스를 탐구한 데이터 분석 프로젝트입니다.

단순한 승패 기록을 넘어, 패스 성공률, 볼 점유율, xG(기대 득점) 등 세부 지표를 활용하여 **데이터 기반의 인사이트(Data-Driven Insights)**를 도출하는 것을 목표로 했습니다.

## 🎯 분석 목표 (Objectives)
* **현대 축구 트렌드 분석**: 높은 점유율이 반드시 승리로 이어지는지(Possession vs Victory) 상관관계 분석
* **팀별 스타일 비교**: 공격 지향적 팀 vs 수비 지향적 팀의 스탯 패턴 시각화
* **주요 선수 퍼포먼스**: 골, 어시스트, 키패스 등을 통한 Top Performer 분석
* **상관관계 도출**: 경기 승패에 가장 큰 영향을 미치는 핵심 변수(Feature) 식별

## 🛠️ 기술 스택 (Tech Stack)
* **Language**: Python 3.x
* **Data Manipulation**: Pandas, NumPy
* **Visualization**: Matplotlib, Seaborn
* **Environment**: Jupyter Notebook 

## 📂 프로젝트 구조 (File Structure)
